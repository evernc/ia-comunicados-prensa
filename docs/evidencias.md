# Evidencias recomendadas

## Práctica 1.1 - SCRUM en Jira

Capturas recomendadas:
1. Proyecto creado en Jira.
2. Product Backlog con Epics.
3. Stories creadas.
4. Tasks creadas.
5. Sprint Planning.
6. Daily Scrum con fechas.
7. Roles Scrum definidos.

## Práctica 1.2 - Git y GitHub

Capturas recomendadas:
1. Comando `git --version`.
2. Comando `git config --list`.
3. Repositorio creado en GitHub.
4. Archivos del proyecto.
5. Commits realizados.
6. Rama secundaria creada.
7. Pull Request creado.
8. Pull Request unido a main/master.

## Práctica 1.3 - Diagrama de Gantt en Jira

Capturas recomendadas:
1. Timeline o cronograma en Jira.
2. Tareas con fechas de inicio y fin.
3. Dependencias entre tareas.
4. Sprint iniciado.
5. Tablero Kanban.
6. Tareas asignadas al Development Team.
7. Tareas en To Do, In Progress y Done.
