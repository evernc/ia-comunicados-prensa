# IA Redactora de Comunicados y Notas de Prensa

## Descripción del proyecto

Este proyecto es un prototipo de inteligencia artificial para redactar comunicados y notas de prensa en segundos a partir de hechos clave.

El usuario ingresa información como título, fecha, lugar, audiencia, hechos importantes, tono de redacción y persona responsable. El sistema genera automáticamente un borrador profesional que puede ser revisado y publicado por el equipo de comunicación.

## Objetivo

Automatizar la creación de borradores de comunicados y notas de prensa para que el equipo solo tenga que revisar, corregir y publicar.

## Funciones principales

- Captura de hechos clave.
- Selección del tipo de texto: comunicado o nota de prensa.
- Selección del tono: formal, institucional, informativo o urgente.
- Generación automática del borrador.
- Métricas básicas del texto generado.
- Botón para copiar el borrador.
- Botón para limpiar el formulario.
- Diseño moderno en modo oscuro.

## Archivos del proyecto

- `index.html`: estructura principal de la interfaz.
- `style.css`: estilos visuales del proyecto.
- `script.js`: lógica para generar el borrador.
- `docs/backlog_scrum.md`: Epics, Stories y Tasks para Jira.
- `docs/daily_scrum.md`: ejemplo de reuniones Daily Scrum.
- `docs/comandos_git.md`: comandos básicos de Git para la práctica.
- `docs/gantt_cronograma.md`: cronograma tipo Gantt para Jira.
- `docs/evidencias.md`: lista de capturas recomendadas.

## Cómo usar

1. Abrir el archivo `index.html` en el navegador.
2. Llenar los datos solicitados.
3. Presionar el botón **Generar borrador**.
4. Revisar el texto generado.
5. Copiar el comunicado final.

## Nota técnica

Este prototipo simula el comportamiento de una IA mediante plantillas dinámicas en JavaScript. Para una versión real, se podría conectar a una API de inteligencia artificial.
